# TechStore - Programming Resources Marketplace

A modern e-commerce website similar to Digikala, specifically designed for programming resources, courses, and digital products. Features dedicated sections for C Programming and JavaScript with a beautiful, responsive design.

## 🚀 Features

### Core Functionality
- **Modern E-commerce Design** - Clean, professional layout inspired by Digikala
- **Dedicated Programming Sections** - Specialized pages for C Programming and JavaScript
- **Responsive Design** - Fully responsive across all devices
- **Interactive Search** - Real-time search functionality with filtering
- **Product Filtering** - Advanced filters by category, level, price, and more
- **Shopping Cart Simulation** - Add to cart and wishlist functionality
- **Newsletter Subscription** - Email subscription with validation

### User Experience
- **Smooth Animations** - Scroll-triggered animations and hover effects
- **Code Snippet Highlighting** - Syntax-highlighted code examples
- **Learning Paths** - Structured learning roadmaps for both technologies
- **Toast Notifications** - User feedback for all interactions
- **Mobile Menu** - Collapsible navigation for mobile devices

### Pages Included
1. **Homepage** (`index.html`) - Overview with featured categories and popular products
2. **C Programming** (`c-programming.html`) - Dedicated C programming resources
3. **JavaScript** (`javascript.html`) - Modern JavaScript frameworks and tools

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with CSS Grid, Flexbox, and animations
- **JavaScript (ES6+)** - Interactive functionality and DOM manipulation
- **Bootstrap Icons** - Professional icon set
- **Google Fonts (Inter)** - Modern typography

## 💡 Customization

### Adding New Products
1. Copy existing product card HTML structure
2. Update product information (title, category, price)
3. Adjust product image/icon as needed
4. Ensure proper classes for filtering

### Modifying Colors
1. Update CSS custom properties in `:root`
2. All colors use CSS variables for consistency
3. Maintain contrast ratios for accessibility

### Adding New Pages
1. Follow existing HTML structure
2. Include proper navigation links
3. Maintain consistent header/footer
4. Add page-specific styles if needed
